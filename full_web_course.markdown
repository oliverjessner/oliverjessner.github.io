# Wie werde ich Web Entwickler? 

Diese Frage wurde mir so oft gestellt, dass ich angefangen habe auf <a>YouTube Full Courses<a href="https://www.youtube.com/c/ProgrammierenlernenmitOli"> dafür zu erstellen. Um Entwickler zu werden muss mann nicht studieren oder eine Ausbildung machen. Tausende Menschen vor dir haben das bereits vorgemacht. Auf meinen YouTube Kanal ist das natürlich komplett kostenlos. 

## HTML5

Der erste Schritt ist es HTML zu lernen. Das kannst du ganz einfach mit meinem 3-teiligen HTML Kurs. In nur 40 Minuten lernst du alles über HTML. Im Vergleich eine Folge Game of Thrones dauert länger.

Nach dem HTML5 full course fährst du mit dem CSS3 Course weiter. CSS ist schon viel komplexer als HTML, aber du lernst die wichtigsten Elemente in 2 Stunden 15 Minuten. Wenn du dir einen Marvel Avengers Film ansiehst, dann musst du mehr Zeit investieren.
